Sobre
 Uma rede social no qual permite que o usuario possa criar posts, fazer comentararios, curtir, seguir e ser seguido por outros usuarios, enviar mensagems privadas.
  
Tecnologias utilizadas
  - React
  - NodeJs
  - Typescript
  - Postgres
  - Redux toolkit
  - Styled-components
  - Express
  - Socket.io
